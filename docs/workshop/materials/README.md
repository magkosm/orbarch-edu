# Workshop materials

Print-and-go resources for the [workshop](../README.md). Everything here is text /
HTML so it lives in git; binary decks (`.pptx`) are added by you (placeholders below).

## What's here

| File | What it is | How to use |
| --- | --- | --- |
| [`qr-codes.html`](./qr-codes.html) | Printable QR sheet for every tool, in EN/SV/EL | Open in a browser → **Print**. Cut out the language section you need. |
| [`slides-outline.md`](./slides-outline.md) | Slide-by-slide deck outline with placeholders | Build your deck from it; export per language. |
| [`student-handout.md`](./student-handout.md) | One-page student worksheet (scores, checklist, sketch box) | Print 1 per student or pair. |

## Print checklist

- [ ] `qr-codes.html` → print the language section(s) for your group.
- [ ] `student-handout.md` → 1 per student/pair (A4).
- [ ] A3 paper, rulers, markers for the sketch phase (1 set per group of 4).
- [ ] Slides loaded on the projector.
- [ ] Room speakers tested with the cabin-noise clip.
